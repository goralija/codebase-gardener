# Repository Constitution

## Product Purpose

Normal Repo demonstrates a small, well-documented Python service.

## Autonomous Fixes Allowed

- Documentation updates.
- Test-only refactors.
- Lint and formatting changes.

## Assisted-Only Fixes

- Public API behavior changes.

## Ignored Paths

- `.venv/**`
- `dist/**`

## Test Rules

Run `pytest` after changing `src/**`.
